# 🗺️ maps

Interaktive Schweizer Geodaten-Tools für den Unterricht – gebaut mit Swisstopo-APIs, Leaflet und reinem HTML.

## Tools

| Tool | Beschreibung |
|------|-------------|
| **Kartenschieber** | Historische Swisstopo-Karten mit Swipe-Slider vergleichen (Dufourkarte 1844 bis heute) |
| **Kartenvergleich** | Zwei synchronisierte Karten nebeneinander – gleicher Ausschnitt, verschiedene Epochen |
| **Naturwege** | Routen auf Swisstopo-Karten manuell zeichnen, Distanz messen, GPX exportieren |
| **3D Flug** | Helikopter-Fluganimation über der Schweiz mit Wegpunkt-Planung und Video-Export |
| **Blockbild** | 3D-Geländedarstellung aus Swisstopo-Höhendaten mit Textur-Overlay |
| **Windrad** ⚙️ | Animierte Windräder massstabsgetreu in Fotos einblenden *(in Arbeit)* |
| **Sonnenstand** ⚙️ | Sonnenstand und Terrain-Verschattung für beliebige Standorte in der Schweiz *(in Arbeit)* |

## Verwendung

Alle Tools sind einzelne HTML-Dateien – kein Build-Schritt, keine Installation, keine Abhängigkeiten. Einfach öffnen und verwenden.

## Datenquellen

- [swisstopo](https://www.swisstopo.admin.ch/) – Höhendaten, Kacheln, Geocoding
- [OpenStreetMap](https://www.openstreetmap.org/) / [OSRM](https://project-osrm.org/) – Strassennetz und Routing
- [Leaflet.js](https://leafletjs.com/) – Kartendarstellung
